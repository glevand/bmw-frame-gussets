# bmw-frame-gussets

Scalable Vector Graphics files for gussets to reinforce the frame of BMW R100GS motorcycles.

Autocad DXF files are here: [Autocad DXF](https://github.com/glevand/bmw-frame-gussets/tree/master/autocad-dxf)
